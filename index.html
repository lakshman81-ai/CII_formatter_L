CAESAR II Parser

A production-ready bi-directional parser and optimization tool for converting CAESAR-II Neutral Files (.cii/.neu) to structured tabular data and back.

Notice: This is a Python-based backend application. It cannot run directly here on GitHub Pages.
How to run the Web Interface locally

To use the full visual web application, run the Flask backend on your local machine:

# 1. Clone the repository
git clone https://github.com/lakshman81-ai/CII_formatter_L.git
cd CII_formatter_L

# 2. Install dependencies
pip install -r requirements.txt

# 3. Launch the web server
python cii_roundtrip.py web

# 4. Open in browser
# http://localhost:5000
Command Line Usage

You can also run the tool directly from the terminal to export and reconstruct files:

python cii_roundtrip.py reconstruct \
--input "original.neu" \
--table "output.csv" \
--out "generated.neu" \
--optimize
